<h2> Project Name : Online Reservation System |Java + Spring Boot + MySQL(task 1)</h2>
<p>Excited to share my latest project — an Online Reservation System developed using Java, Spring Boot, Spring Data JPA, and MySQL.

This project is a console-based railway reservation management system that allows users to register, log in, reserve train tickets, and cancel reservations using PNR numbers. The application follows a layered architecture using Controller, Service, Repository, and Entity components for better code organization and maintainability.</p>
<h2>🔹 Key Features:</h2>
<ul>
  <li> User Registration & Login Authentication</li>
<li>Train Ticket Reservation System</li>
<li>PNR-based Reservation Cancellation</li>
<li>MySQL Database Integration</li>
<li>CRUD Operations using Spring Data JPA</li>
<li>Auto-generated PNR Numbers</li>
<li>Layered Architecture Implementation</li>
